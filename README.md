# p5.js Clock Project

A simple clock display project created with p5.js.

## Features

- Real-time display of current time (hours, minutes, seconds)
- Graphics rendering using p5.js library

## Usage

Simply open `index.html` in your browser to run the project.

## File Structure

- `index.html` - Main HTML file
- `sketch.js` - p5.js drawing code
- `p5.js` / `p5.min.js` - p5.js library files
